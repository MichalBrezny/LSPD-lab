# Land Surface Processes and Dynamics - Laboratory Notebooks

This repository contains Jupyter notebooks for laboratory exercises in Land Surface Processes and Dynamics (LSPD). The notebooks are designed to be run in Google Colab or locally with Jupyter.

## üìö Lab Contents

### Lab 1: Introduction to Land Surface Processes and Dynamics
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/MichalBrezny/LSPD-lab/blob/main/labs/Lab01_Introduction_to_LSPD.ipynb)

Learn the fundamentals of:
- Surface energy balance components
- Diurnal cycles of energy fluxes
- Temperature and moisture variations
- Energy balance closure and Bowen ratio

### Lab 2: Energy Balance and Surface Temperature
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/MichalBrezny/LSPD-lab/blob/main/labs/Lab02_Energy_Balance_Temperature.ipynb)

Explore:
- Radiation components (shortwave and longwave)
- Effect of surface albedo on energy balance
- Cloud cover impacts on radiation
- Surface temperature calculations

### Lab 3: Soil Moisture and Hydrology
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/MichalBrezny/LSPD-lab/blob/main/labs/Lab03_Soil_Moisture_Hydrology.ipynb)

Study:
- Soil water retention curves
- Infiltration and drainage processes
- Soil moisture dynamics
- Water balance components

### Lab 4: Vegetation and Evapotranspiration
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/MichalBrezny/LSPD-lab/blob/main/labs/Lab04_Vegetation_Evapotranspiration.ipynb)

Understand:
- Penman-Monteith equation for ET
- Stomatal conductance and controls
- Vegetation effects on water and energy fluxes
- Plant-atmosphere interactions

### Lab 5: Surface Fluxes and Atmospheric Interactions
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/MichalBrezny/LSPD-lab/blob/main/labs/Lab05_Surface_Fluxes_Atmospheric_Interactions.ipynb)

Integrate concepts:
- Complete energy balance
- Turbulent flux calculations
- Atmospheric stability and Monin-Obukhov theory
- Boundary layer development
- Land-atmosphere feedbacks

## üöÄ Getting Started

### Option 1: Google Colab (Recommended)
Simply click on any of the "Open in Colab" badges above. All dependencies are pre-installed in Colab.

### Option 2: Local Installation

1. Clone this repository:
```bash
git clone https://github.com/MichalBrezny/LSPD-lab.git
cd LSPD-lab
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Start Jupyter:
```bash
jupyter notebook
```

4. Navigate to the `labs/` directory and open any notebook.
